# Консольные утилиты для агента

Там, где задачу надо решать в рантайме, агент вызывает обычные консольные инструменты сам. Быстрые аналоги классических тулов ускоряют цикл разработки в разы.

## Git-хостинг

- gh - GitHub из консоли: PR, issues, релизы, CI - https://github.com/cli/cli
- glab - GitLab из консоли, то же самое для GitLab - https://gitlab.com/gitlab-org/cli

## Поиск и навигация по коду

- ripgrep (rg) - быстрый поиск по коду, уважает gitignore - https://github.com/BurntSushi/ripgrep
- fd - быстрый поиск файлов вместо find - https://github.com/sharkdp/fd
- ast-grep (sg) - структурный поиск и рефакторинг по AST, а не регекспом - https://github.com/ast-grep/ast-grep
- fzf - интерактивный фильтр по спискам - https://github.com/junegunn/fzf

## Данные и запуск

- jq - фильтрация JSON, пока он не съел контекст - https://github.com/jqlang/jq
- yq - то же самое для YAML - https://github.com/mikefarah/yq
- curl - HTTP-запросы - https://github.com/curl/curl
- docker - запуск и проверка в контейнерах - https://www.docker.com
