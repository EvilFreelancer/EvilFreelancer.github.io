# Скилы для агентной разработки

Подборка скилов, которые пригодятся кодовому агенту. Сначала мои, дальше общие практики и коллекции. Список открытый, дополняется.

## Мои скилы (rpa-skills)

Весь каталог ставится одной командой: https://github.com/EvilFreelancer/rpa-skills

- rpa-init - онбординг агента на репозиторий: изучает код, документацию и тесты, поднимает окружение, прогоняет тесты, пишет короткий отчёт - https://github.com/EvilFreelancer/rpa-init
- rpa-feat - разработка фичи строго по BDD: план, красные тесты, реализация до зелёных, полный прогон, документация, линтер - https://github.com/EvilFreelancer/rpa-feat
- rpa-bugfix - починка бага: сначала воспроизводящий тест, потом фикс, потом полный прогон и отчёт - https://github.com/EvilFreelancer/rpa-bugfix
- rpa-gen-rules - генерация и синхронизация правил проекта под Cursor, Claude Code и AGENTS.md - https://github.com/EvilFreelancer/rpa-gen-rules
- logika - формальная логика по Челпанову: проверка рассуждений и поиск логических ошибок - https://github.com/EvilFreelancer/logika
- token-cost - расчёт себестоимости токена на своём железе, электричество плюс амортизация - https://github.com/EvilFreelancer/token-cost
- mikrotik-config-gen - генерация и ревью конфигов MikroTik RouterOS - https://github.com/EvilFreelancer/mikrotik-config-gen

## Общие инженерные практики

Большая коллекция скилов superpowers: https://github.com/obra/superpowers

- test-driven-development - тесты до кода, красный, зелёный, рефактор
- systematic-debugging - гипотеза, воспроизведение, фикс вместо тыка наугад
- brainstorming - проработка задачи до реализации
- writing-plans и executing-plans - план под многошаговую работу и его выполнение по чекпоинтам
- requesting-code-review и receiving-code-review - запрос ревью и работа с замечаниями
- verification-before-completion - проверка результата перед словом готово
- using-git-worktrees - изоляция параллельной работы
- subagent-driven-development - оркестрация через сабагентов
